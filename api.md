
## Api请求说明:
- 请求支持GET,POST

#### 请求格式 : http://hostname/mapi_v2/service/method
#### Example : http://trymob.edusoho.cn/mapi_v2/Category/getAllCategories
#### 备注:
* hostname  域名
* service   服务接口名
* method    接口方法
* 没有标注的方法，参数都是可选
